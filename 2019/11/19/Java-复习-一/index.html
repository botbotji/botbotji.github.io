Jdk（java开发工具包）

  • 安装目录：
    1. bin（java的可执行程序）
    2. DB (内置小型数据库)
    3. Jre(java运行环境)
    4. Lib(java的类库)

安装完成需要配置环境变量（都是目录，控制台执行时需要引用）：

  1. JAVA_HOME: C:\Program Files\Java\jdk1.8.0_162(JDK安装的主目录)
  2. Path：C:\Program Files\Java\jdk1.8.0_162\bin（可执行程序的目录，在控制台任意目录下启动）
  3. CLASSPATH: C:\Program Files\Java\jdk1.8.0_162\lib;.（需要调用的类的目录）

  • 可能提示的问题：
    找不到或无法加载主类：查看classpath变量是否配置正确

  • Eclipse安装：注意jdk版本和eclipse版本相同（都是32位或64位）
