Docker

  • Docker是一个开源的应用容器引擎；
  • Docker支持将软件编译成一个镜像；然后 在镜像中各种软件做好配置，
    将镜像发布 出去，其他使用者可以直接使用这个镜像。 运行中的这个镜像称为容器，容器启动是 非常快速的

    核心概念

  • docker主机：安装了Docker程序的机器(Docker直接安装在系统上面)
  • docker客户端(Client):连接Docker主机进行操作 API与Docker守护进程通行
  • docker仓库(Regjistries):Docker仓库用来保存镜像，可以理解为代码控制中的代码仓库。
  • docker镜像(Images):软件打包好的镜像；放在docker仓库中；
  • dovker容器(Container):镜像启动后的实例称为一个容器:容器时独立运行的 一个人或一组应用；
  • 使用Daocker步骤：
    • 安装Docker
    • 去Docker仓库找到这个软件的镜像
    • 直接使用Docker运行这个镜像，这个镜像就会生成一个Docker容器
    • 对容器的启动停止就是对软件的启动停止

        xxx-starter —》xxxx-starter-autoconfigurer
